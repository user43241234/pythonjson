This is an easy-to-access json file data writer. The install instruction are below. You will need the python app for all of this.
Windows: Double-click the installer and follow the install instructions. Once finished, place the .zip file into an empty folder and extract all the files. Then double click pg.py.
Linux & Mac: Open terminal and type `sudo apt install Wine`. Then, type `Wine pg.py.exe`. After isntalling the program, extract the .zip file contents and run pg.py. You will need the python app.
